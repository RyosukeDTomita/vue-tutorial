## ENVIRONMENT

- Vue projectの作成

```shell
npm create vue@latest # このリポジトリでは事前に実施済み
```
- npmで実行

```shell
cd vue-tutorial
npm install
npm run dev # http://localhost:5173で実行されている
```
- build

```shell
npm run build
```
